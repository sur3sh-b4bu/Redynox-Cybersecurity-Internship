# Redynox Cybersecurity Internship

This repository contains weekly reports submitted as part of the Redynox Cybersecurity Internship.

## Internship Focus
- Vulnerability Assessment
- Web Application Security
- Ethical Hacking Practices
- Security Reporting

## Repository Structure
- Reports/WEEK-01 – Initial assessment and reconnaissance
- Reports/WEEK-02 – Vulnerability identification and analysis
- Reports/WEEK-03 – Exploitation and validation
- Reports/WEEK-04 – Final report and remediation insights
